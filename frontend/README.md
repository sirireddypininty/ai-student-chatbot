# AI Student Chatbot

A simple chatbot built using Spring Boot, HTML, CSS and JavaScript.

## Technologies Used

* Java
* Spring Boot
* Maven
* HTML
* CSS
* JavaScript

## How to Run

1. Open terminal.
2. Go to backend folder.

```bash
cd backend
mvn spring-boot:run
```

3. Open frontend/index.html in browser.

## Features

* Student chatbot
* Answers questions about Java, DSA, DBMS, OOPS
* Placement and interview tips

## Future Improvements
